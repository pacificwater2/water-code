# Changelog

## v0.1.0 - 2026-04-05

Initial public-ready Water Code release candidate.

- built a clean-room terminal coding agent runtime with CLI, REPL, sessions, tools, and providers
- added diff/patch editing, project context loading, `WATER.md`, and richer permission modes
- added custom commands, agents, skills, plugins, MCP, background tasks, and multi-agent orchestration
- added HTTP bridge server/client, adapter protocol, streamed prompt events, and VS Code shim support
- added Git/worktree-aware project switching, remote session handling, doctor/onboarding flows, and approval history
- added package/install validation, bridge smoke tests, and broader automated regression coverage
